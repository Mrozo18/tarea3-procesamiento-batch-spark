# Tarea 3 - Procesamiento de Datos con Apache Spark

Este repositorio contiene el desarrollo del procesamiento batch realizado con Apache Spark mediante PySpark en Google Colab.

## Descripción del proyecto

El proyecto analiza un conjunto de datos propio de ventas compuesto por 10.000 registros simulados. El objetivo es identificar productos, categorías, ciudades y meses con mayor impacto en los ingresos.

## Herramientas utilizadas

- Google Colab
- Python
- PySpark
- Apache Spark
- Pandas
- Matplotlib

## Procesamiento realizado

El notebook desarrolla las siguientes etapas:

1. Instalación de PySpark.
2. Creación de una sesión Spark.
3. Generación de un conjunto de datos de ventas.
4. Carga del archivo CSV mediante Spark.
5. Limpieza de datos.
6. Transformación de datos.
7. Análisis exploratorio por producto, categoría, ciudad y mes.
8. Visualización de resultados mediante gráficas.

## Archivo principal

- `Tarea3_Procesamiento_Batch_Spark.ipynb`

## Ejecución

Para ejecutar el proyecto, abrir el notebook en Google Colab y ejecutar las celdas en orden desde la primera hasta la última.
